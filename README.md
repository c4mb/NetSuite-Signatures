# NetSuite Signatures

Alpha Vital email signatures for NetSuite users.

---

## Setup

### 1. Open Preferences

Navigate to **Home → Set Preferences**

<img width="820" height="711" alt="Screenshot-1" src="https://github.com/user-attachments/assets/135efff5-bc84-4894-8672-afd153475b6f" />

### 2. Add Your Signature

Paste the signature code into the **Signature** field, then check **Add Signature to Messages**.
