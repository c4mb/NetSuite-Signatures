# NetSuite Signatures

Alpha Vital email signatures for NetSuite users.

---

## Setup

### 1. Locate your .html file above 👆

Open your .html file and **copy the code block**

(Warehouse staff will all use Warehouse-Staff.html)

### 2. Login to NetSuite

Navigate to **Home → Set Preferences**

<img width="2226" height="1800" alt="Image" src="https://github.com/user-attachments/assets/bd100201-b210-4367-9a48-152abfc1a041" />

### 3. Add Your Signature

Paste the signature code into the **Signature** field, then check **Add Signature to Messages** and click **save**.
